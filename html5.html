Aide - Générateur de carroyage CADO

À propos du générateur de carroyage

Cet outil permet de générer un carroyage géographique à partir d'un point de référence. Le carroyage est composé de lignes horizontales et verticales formant une grille, avec des points centraux nommés par une combinaison de lettre et de nombre.

Instructions

  1. Entrez les coordonnées dans l'un des formats disponibles (décimal, DMS, ou Plus Code).
  2. Configurez le carroyage en définissant l'échelle (distance entre les lignes), la couleur, la transparence, etc.
  3. Choisissez une grille prédéfinie ou personnalisée.
  4. Sélectionnez le format de sortie souhaité (KML, KMZ, GeoJSON ou GPX).
  5. Cliquez sur GÉNÉRER pour créer et télécharger le fichier.

Grilles prédéfinies

  • Z26 : grille 26x26 (de A à Z pour les colonnes, de 1 à 26 pour les lignes)
  • Z14 : grille 26x14 (de A à Z pour les colonnes, de 1 à 14 pour les lignes)
  • Q9 : grille 17x9 (de A à Q pour les colonnes, de 1 à 9 pour les lignes)

Grille personnalisée

Vous pouvez définir une grille personnalisée en spécifiant :

  • Les lignes : de -26 à 52
  • Les colonnes : de -Z à AZ (notation par lettres)

Formats de fichiers

  • KML/KMZ : Pour Google Earth et Google Maps
  • GeoJSON : Format standard pour les données géographiques
  • GPX : Format compatible avec OSMAND et autres applications GPS

Notation négative

Pour les nombres négatifs, utilisez simplement le signe "-" (ex: -5).

Pour les lettres négatives, utilisez également le signe "-" (ex: -A, qui précède A).
